[프로젝트 현황 + 시험항목/문제항목 탭 화면 디자인 통합 정리]

목표:
- 프로젝트 현황 화면과 시험항목/문제항목 탭 화면의
  타이포그래피(글자 크기/굵기), 제목 규칙, 여백을 하나의 기준으로 통일한다.
- 기능, 데이터 로직, 레이아웃 구조는 변경하지 않고
  시각적 통일성만 개선한다.

==================================================
1) 공통 타이포그래피 규칙 (전 화면 공통)
==================================================

[Level 1] 화면/카드의 최상위 제목
- 대상:
  · 프로젝트 현황
  · 시험항목/문제항목 카드의 상단 제목
- font-size: 18~20px
- font-weight: semibold 이상

--------------------------------------------------
[Level 2] 섹션 제목
--------------------------------------------------
- 대상:
  · 프로젝트 현황 내 섹션 제목 (일정, 제품사양)
  · 시험항목/문제항목 탭의 섹션 제목
    (일정 관리, 시험 조건, 판정 기준, 시험 데이터,
     시험 사진, 시험 그래프, 첨부파일 등)
- font-size: 15~16px
- font-weight: semibold
- 좌/우 컬럼 동일한 스타일 적용

※ 프로젝트 현황 섹션 제목 앞의 번호는 제거
  ("1. 일정" → "일정", "2. 제품사양" → "제품사양")
※ 시험항목/문제항목 내부의 항목 번호(1), 2), 3) 등)는 유지

--------------------------------------------------
[Level 3] 필드 라벨 / 상태 정보
--------------------------------------------------
- 대상:
  · 시험 진행율, 시험 합격율, 문제 개선율, 보고서 작성율
  · 날짜 라벨(시작일, 완료예정일, 실제완료일, 최종 수정일)
  · 상태 선택(진행 상태, 시험 결과, 보고서)
- font-size: 13~14px
- font-weight: regular
- 기본 텍스트보다 한 단계 연한 색상

--------------------------------------------------
[Level 4] 보조 텍스트 / placeholder
--------------------------------------------------
- 대상:
  · "설명 추가…"
  · "입력하세요"
  · "클릭하여 업로드"
- font-size: 12px
- 연한 회색

==================================================
2) 간격(Spacing) 규칙 (전 화면 공통)
==================================================
- 섹션 제목과 내용 사이 여백: 12~16px
- 입력 필드 간 수직 간격: 균일하게 유지
- 프로젝트 현황 요약 영역과 하단 섹션(일정/제품사양) 사이에
  시각적으로 구분되는 여백을 둔다.

==================================================
3) 프로젝트 현황 화면 적용 사항
==================================================
- "프로젝트 현황" 제목은 Level 1 규칙 적용
- OK / NG / TBD 요약 및 진행율 수치는 Level 2 규칙 적용
- 진행율 라벨(시험 진행율, 시험 합격율 등)은 Level 3 규칙 적용
- 섹션 제목의 번호는 제거한다.

==================================================
4) 시험항목 / 문제항목 탭 화면 적용 사항
==================================================
- 카드 상단의 항목 번호(예: 11)는
  섹션 제목보다 작은 크기로 표시하여
  제목처럼 보이지 않도록 한다.
- '최종 수정일'은 보조 정보로 취급하여 Level 3 스타일 적용
- 좌측(텍스트 입력)과 우측(업로드 영역)의
  섹션 제목 스타일을 완전히 동일하게 맞춘다.

==================================================
5) 변경 범위 제한 (중요)
==================================================
- 글자 크기, 굵기, 색상 톤, 여백, 제목 텍스트(번호 제거)만 수정한다.
- 컴포넌트 위치, 레이아웃 구조, 기능, 저장/업로드/필터 로직은
  변경하지 않는다.
